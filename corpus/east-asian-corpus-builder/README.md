# Usage

## Install dependencies

```bash
pip install -r requirements.txt
```

## Run the script

```bash
python build_east_asian_corpus_v9_singlefile.py --seeds seeds_east_asian.json --outdir output
```

## Output files

After running, the output directory will contain files such as:

- `east_asian_corpus_v9_singlefile.jsonl`
- `east_asian_chunks_v9_singlefile.jsonl`
- `section_scoring_log_v9_singlefile.jsonl`
- `metadata_v9_singlefile.csv`
- `summary_v9_singlefile.json`
